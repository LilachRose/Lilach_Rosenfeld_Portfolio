UPLOAD INSTRUCTIONS - IMPORTANT

Before uploading this version to GitHub:
1. Delete every existing file in the repository.
2. Upload ONLY these two items to the repository ROOT:
   - index.html
   - assets folder

Do NOT upload the ZIP file itself.
Do NOT upload files from inside the assets folder into the root.
Do NOT rename any image to index.html.
Do NOT add vercel.json.
Do NOT add app.js, portfolioData.js, or styles.css.

After upload, GitHub root must show exactly:
- index.html
- assets/
- README_UPLOAD_FIRST.txt (optional)

If Vercel shows text starting with JFIF, it means a JPG image was uploaded or served as the home page instead of index.html.
